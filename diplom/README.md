## Сборка Microsoft Word (.docx)

После установки зависимостей (один раз):

```bash
cd /Users/floater/Desktop/курсовая
python3 -m venv .venv-docx
.venv-docx/bin/pip install python-docx
.venv-docx/bin/python diplom/build_docx.py
```

Файл создаётся по пути **`diplom/VKR_Symbolicheskaya_regressiya_chernovik.docx`**. В начало главы 4 добавляются таблицы по актуальному **`kursovaia/kursovaia/result.txt`**; полный текст протокола — в приложении Д.

## Порядок работы

1. Утвердить титул и ФИО с научным руководителем.
2. Перенести разделы в Word; выставить стил заголовков (по методичке).
3. Проставить **реальные номера страниц** в содержании после вёрстки.
4. Заполнить таблицы главы 4 актуальными значениями из `result.txt` после прогона программы.
5. Проверить список литературы у нормоконтролёра.
