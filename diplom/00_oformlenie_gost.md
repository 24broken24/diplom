# Рекомендации по оформлению ВКР (ориентиры)

Ниже приведены распространённые требования российских вузов; **исполнение уточняется по методическим указаниям КубГУ и кафедры**.

## Текст основной части

- Шрифт: **Times New Roman**, кегль **14 pt**, межстрочный интервал **1,5**.
- Поля: чаще всего левое **30 мм**, правое **15 мм**, верх/низ **20 мм** (проверить по методичке).
- Абзацный отступ **1,25 см**. Выравнивание по ширине.
- Нумерация страниц: арабские цифры внизу по центру; титульный лист обычно **не нумеруют** или считают без отображения (по правилам кафедры).

## Заголовки

- **Глава** — с новой страницы, прописными, по центру (или по методичке: с абзаца).
- **1.1, 1.2** — полужирным или по шаблону кафедры.
- Точка в конце заголовка — по методичке (часто **не ставят**).

## Формулы и рисунки

- Формулы нумеруются в круглых скобках справа, если на них есть ссылки в тексте.
- Рисунки: **«Рисунок 1.2 – Название»**; подпись размещают под рисунком (по ГОСТ 2.105–95 для схем в техработах; для ВКР — по методичке университета).
- Таблицы: **«Таблица 2.1 – Название»**; заголовок над таблицей (обычно).

## Список литературы

Оформлять по **ГОСТ Р 7.0.100–2018** (библиографическая запись) или уточнённому на кафедре шаблону. Электронные ресурсы: заголовок, URL, дата обращения.

## Сокращения

При первом появлении: полная форма и сокращение в скобках, далее допускается сокращение (например, *среднеквадратичная ошибка (СКО, RMSE)*).
